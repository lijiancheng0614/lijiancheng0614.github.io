ASP.NET发布到Azure VM

May 07, 2015 in Develop / Tagged in ASP.NET, Azure, C#
  1. 环境
  2. 步骤
  3. 参考

主要使用ASP.NET新建一个工程，并将其打包deploy到Azure Virtual Machine中。

环境

  • 支持ASP.NET的Visual Studio。

  • Windows Features中ASP.NET功能和IIS功能已打开的Azure Virtual Machine。

步骤

  1. 打开Visual Studio，新建一个ASP.NET Web Application工程。

  2. 选择工程，右键选Pulish...。

    • Profile选项卡中选择Custom，弹出来一个对话框填Profile的名称，随便填一个，OK。

    • Connection选项卡中的Publish method:选File System，Target location:随便填一个本地路径就好，如C:\MyAppFolder，Next。

    • Setting选项卡根据个人需要，可以直接Next。

    • Preview选项卡点Publish即可。然后就可以在本地相应路径中看到发布的文件。

  3. 把发布的整个文件夹复制到Azure VM中的wwwroot位置，然后打开IIS控制面板，在左侧面板中刷新，看到相应的文件夹。右键选Convert to Application即可。
